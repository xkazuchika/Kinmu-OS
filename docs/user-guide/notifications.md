# アプリ内通知

対象バージョン: 0.5.0

## 通知される操作

- 従業員が残業・休日出勤を提出すると、同じ組織の所有者と労務管理者へ通知します。
- 申請が承認・却下されると、申請した従業員へ通知します。
- 審査待ち申請が取り消されると、所有者と労務管理者へ通知します。

通知には業務に必要な最小限の要約だけを保存します。メールや外部サービスへの送信は行いません。

## 未読・既読と対象を開く

サイドメニューの「通知」に未読件数が表示されます。通知を選ぶと既読になり、現在の役割を再確認して本人申請または管理者審査へ移動します。「表示中をすべて既読」で現在読み込んだ未読通知をまとめて既読にできます。

通知後に役割が変わった、対象が現在の権限外になった、または対象状態が変わった場合は、通知を既読にして安全な案内を表示します。保存済みの任意URLへ移動することはありません。

## 現行版の制限

メール、Slack等への外部通知、時間単位休暇、自動法令判定、多段承認、管理者による代理申請には対応していません。申請・承認は法令や就業規則への適合を判定するものではありません。
