# 打刻・勤務実績・プロフィール

対象バージョン: 0.3.0

## 対象役割と開始画面

対象役割は従業員です。ログイン後のホームから当日の打刻を行い、「勤務実績」と「プロフィール」で自分の情報を確認します。事前に管理者が従業員台帳と利用者を紐付けている必要があります。

## 出勤・退勤・休憩

1. 業務開始時に「出勤する」を選びます。
2. 休憩開始時に「休憩を開始」、再開時に「休憩を終了」を選びます。
3. 業務終了時に「退勤する」を選びます。
4. ホームの状態と時刻が変わったことを確認します。

打刻は出勤、休憩開始、休憩終了、退勤の順序で行います。二重出勤や、出勤前の退勤など不正な順序は保存されません。

## 勤務実績

1. 「勤務実績」を開きます。
2. 表示する月を選び、日ごとの打刻、実労働、残業を確認します。
3. 誤りがある場合は「修正を申請」を選びます。

詳しい申請方法は[勤怠修正の申請と審査](attendance-corrections.md)を参照してください。

## プロフィール

「プロフィール」では社員番号、所属、雇用区分などを確認し、連絡先メールアドレスと電話番号を更新できます。氏名や所属の変更は管理者へ依頼してください。
