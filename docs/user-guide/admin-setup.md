# 初期設定と従業員管理

対象バージョン: 0.5.0

## 対象役割と開始画面

対象役割は所有者と労務管理者です。最初の組織と所有者は `/setup` から作成し、その後はホームから各管理画面へ進みます。

## 初期設定

1. 組織名、所有者名、所有者メールアドレス、タイムゾーンを入力します。
2. 発行された設定リンクを開き、12文字以上のパスワードを設定します。
3. ログイン後、ホームが表示されることを確認します。

## 利用者管理

1. 「利用者管理」を開き、氏名、メールアドレス、役割を入力します。
2. 設定リンクを本人へ安全な経路で渡します。
3. 必要に応じて役割変更または無効化を行います。

## 部署と従業員台帳

1. 「従業員」から「部署を管理」を開き、部署コードと部署名を登録します。
2. 従業員を追加し、社員番号、氏名、雇用区分、入社日、所属部署を入力します。
3. 従業員詳細でログイン利用者を紐付けます。紐付け後、本人が勤務実績とプロフィールを利用できます。

## 勤務ルール

1. 「勤怠」から「勤務ルール」を開きます。
2. 名称、適用開始日、所定開始・終了時刻、休憩時間、1日の所定労働時間を入力します。
3. 組織共通ルール、または特定従業員のルールとして保存します。

完了後は[打刻・勤務実績・プロフィール](employee-attendance.md)で従業員の日常操作を確認してください。

勤務ルールの後に[勤務カレンダーの設定](work-calendar.md)と[休暇種別・付与・残高管理](leave-management.md)を進めます。

## 現行版の制限

時間単位休暇、自動法令判定、多段承認、管理者による代理申請には対応していません。アプリ内通知は残業・休日出勤申請のイベントが対象です。集計と申請承認は法令や就業規則への適合を判定するものではありません。
