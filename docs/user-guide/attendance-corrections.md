# 勤怠修正の申請と審査

対象バージョン: 0.4.0

## 対象役割と開始画面

従業員は「勤務実績」から申請します。所有者と労務管理者は「勤怠申請」から審査します。

## 従業員が申請する

1. 勤務実績で対象日を選び、「修正を申請」を開きます。
2. 打刻を追加するか、既存の種類・時刻を変更または削除します。
3. 修正理由を入力し、差分を確認して「この内容で申請」を選びます。
4. 状態が「審査待ち」になったことを確認します。

審査待ちの申請は「申請を取り消す」で取消できます。申請中は同じ日の新しい申請を重ねて作成できません。

## 管理者が審査する

1. 「勤怠申請」を開き、審査待ちの申請を選びます。
2. 修正前と修正後の差分、申請理由を確認します。
3. 承認する場合は確認画面を経て承認します。却下する場合はコメントを入力します。
4. 状態と監査ログが更新されたことを確認します。

## 競合と再申請

申請後に通常打刻が変わると、古い内容のまま承認できません。従業員は審査待ち申請を取り消し、最新の勤務実績を確認してから再申請します。

## 承認後の集計

承認された内容は有効な打刻に反映され、勤務カレンダー、休暇、欠勤を含む日次状態、実労働、残業、勤怠CSVが同じ結果で再集計されます。締め済み月は申請・取消・審査ができないため、管理者が理由を記録して月を再開してから操作します。

## 現行版の制限

時間単位休暇、自動法令判定、多段承認、管理者による代理申請、通知は未対応です。残業集計と休暇一覧は法令や就業規則への適合を判定するものではありません。
