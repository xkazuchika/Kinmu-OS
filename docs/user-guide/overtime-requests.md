# 残業・休日出勤申請

対象バージョン: 0.5.0

## 申請する

1. 「残業・休日出勤」を開き、現在の入力単位、事前申請、許容差異を確認します。
2. 勤務日、予定開始・終了、予定休憩、理由を入力します。終了が開始以前なら翌日として扱います。
3. 「勤務予定と申請分数を確認」を選び、勤務カレンダーから判定された残業・休日出勤区分、勤務予定、予定分数を確認します。
4. 内容が正しければ確認ダイアログから送信します。

入力時刻は設定された単位に合わせます。事前申請が必須の場合、予定開始後は申請できません。同じ時間帯の審査待ち・承認済み申請、24時間を超える範囲、休憩が時間帯以上の申請も受け付けません。

## 取消と審査結果

申請履歴では審査待ち、承認済み、却下、取消済みを確認できます。自分で取り消せるのは最新状態が審査待ちの申請だけです。却下時は管理者が入力した理由も表示されます。締め済み月の提出・取消・審査はできないため、修正が必要なら管理者へ月次勤怠の再開を依頼します。

## 実績差異を確認する

承認された予定分数は勤怠実績へ加算されません。「勤務実績」で申請分、実績比較分、差分と次の状態を確認します。

- 申請内: 許容分数の範囲内です。
- 申請未満: 実績が申請より少ない状態です。
- 申請超過: 実績が申請と許容分数を超えています。
- 実績なし: 承認済み申請に対応する実績がありません。
- 未申請の実績: 実績はありますが承認済み申請がありません。

申請の審査結果は[アプリ内通知](notifications.md)、勤務実績の修正は[勤怠修正の申請と審査](attendance-corrections.md)も参照してください。

## 現行版の制限

時間単位休暇、自動法令判定、多段承認、管理者による代理申請には対応していません。申請・承認は36協定、法定休日労働、割増賃金などの法令や就業規則への適合を判定・保証するものではありません。
