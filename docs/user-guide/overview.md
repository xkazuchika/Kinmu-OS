# 機能一覧と役割

対象バージョン: 0.3.0

## このガイドについて

Kinmu-OSは、従業員100名以下の組織を対象にしたセルフホスト型の労務・勤怠管理ソフトです。このガイドは所有者、労務管理者、従業員が現在利用できる機能と操作を説明します。

## 役割と権限

- 所有者: 初期設定、利用者、従業員、部署、勤務ルール、勤怠、申請審査、レポート、監査ログを管理できます。
- 労務管理者: 所有者と同じ日常管理機能を利用できます。
- 従業員: 自分の打刻、勤務実績、勤怠修正申請、プロフィールを利用できます。

## 機能一覧

- 組織の初期設定とログイン利用者の管理
- 部署と従業員台帳、利用者との紐付け
- 所定開始・終了時刻、休憩、所定労働時間の勤務ルール
- 出勤、退勤、休憩開始、休憩終了の打刻
- 実労働時間と残業時間の集計
- 勤怠修正の申請、取消、承認、却下
- 未退勤と審査待ちを確認する月次締め、理由付き再開、再締め
- 従業員CSVと勤怠CSVの出力
- 重要操作の監査ログ

詳しい導入手順は[初期設定と従業員管理](admin-setup.md)、日々の操作は[打刻・勤務実績・プロフィール](employee-attendance.md)を参照してください。

## 現行版の制限

- 多段承認は未対応です。
- 管理者による代理申請は未対応です。
- 残業集計は勤務ルールとの差分を表示するもので、労働法令や就業規則への適合を自動判定しません。
