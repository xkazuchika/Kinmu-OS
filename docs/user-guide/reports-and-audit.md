# レポート・CSV・監査ログ

対象バージョン: 0.2.0

## 対象役割と開始画面

対象役割は所有者と労務管理者です。「勤怠」「レポート」「監査ログ」から確認します。

## 勤怠一覧と残業集計

1. 「勤怠」で対象月を選びます。
2. 従業員ごとの実労働、残業、未退勤を確認します。
3. 必要な修正は従業員へ申請を依頼し、承認後に再集計を確認します。

残業時間は適用中の勤務ルールに対する日単位の差分です。法令上の時間外労働、休日労働、割増賃金、36協定への適合を自動判定するものではありません。

## CSV出力

1. 「レポート」を開きます。
2. 従業員CSVまたは勤怠CSVを選びます。
3. 勤怠CSVでは対象月を指定して出力します。
4. UTF-8のCSVを表計算ソフトで開き、件数と対象月を確認します。

## 監査ログ

「監査ログ」では利用者、従業員、打刻、修正申請、承認・却下などの重要操作を時系列で確認できます。調査時は操作日時、操作者、対象、操作内容を照合してください。

不明な結果がある場合は[トラブル対処](troubleshooting.md)を参照してください。
