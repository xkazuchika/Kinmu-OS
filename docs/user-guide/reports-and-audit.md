# レポート・CSV・監査ログ

対象バージョン: 0.4.0

## 対象役割と開始画面

対象役割は所有者と労務管理者です。「勤怠」「レポート」「監査ログ」から確認します。

## 勤怠一覧と残業集計

1. 「勤怠」で対象月を選びます。
2. 従業員ごとの実労働、残業、未退勤を確認します。
3. 必要な修正は従業員へ申請を依頼し、承認後に再集計を確認します。

残業時間は適用中の勤務ルールに対する日単位の差分です。法令上の時間外労働、休日労働、割増賃金、36協定への適合を自動判定するものではありません。

## CSV出力

1. 「レポート」を開きます。
2. 従業員CSV、勤怠CSV、休暇台帳CSVのいずれかを選びます。
3. 勤怠CSVでは対象月を指定して出力します。
4. UTF-8のCSVを表計算ソフトで開き、件数と対象月を確認します。

締め済み月の勤怠CSVは確定スナップショットから作成され、日区分、カレンダー根拠、休暇、欠勤、「月次状態」「締め日時」「締めリビジョン」を含みます。休暇台帳CSVは付与・調整・消化・戻し・失効と残高を含みます。給与連携に使ったCSVは、対象月とリビジョンが分かる名前で安全な保管先へ保存してください。

## 監査ログ

「監査ログ」では利用者、従業員、勤務カレンダー、休暇種別、付与・調整、休暇申請・審査、欠勤、CSV取込・出力、打刻、勤怠修正、月次締め、再開、再締めなどの重要操作を時系列で確認できます。調査時は操作日時、操作者、対象月、リビジョン、操作内容を照合してください。

不明な結果がある場合は[トラブル対処](troubleshooting.md)を参照してください。

## 現行版の制限

時間単位休暇、自動法令判定、多段承認、管理者による代理申請、通知は未対応です。残業集計と休暇一覧は法令や就業規則への適合を判定するものではありません。
